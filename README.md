#  Cheeku Store

##  Project Description

Cheeku Store is a web application that allows users to browse products using a public API. Users can search, filter, and sort products dynamically, providing a smooth and interactive shopping-like experience.

---

##  Features

*  Search products (with debouncing)
*  Filter products by category
*  Sort products by price
*  Add/remove favorites (stored in local storage)
*  Fully responsive design

---

## API Used

Fake Store API
https://fakestoreapi.com/products

---

##  Technologies Used

* HTML
* CSS
* JavaScript (Fetch API)
* Array Higher Order Functions (map, filter, sort)
